# Property Monitor

Baton Rouge Property Tracking System

## Files in This Package

- `app.py` - Main web application
- `monitor_service.py` - Property monitoring logic
- `baton_rouge_scraper.py` - Data fetching from EBR APIs
- `requirements.txt` - Python dependencies
- `data/` - Data storage folder

## How to Upload to GitHub

1. Go to: https://github.com/thewh0letruth/property.monitor
2. Click "Add file" → "Upload files"
3. Drag all files from this folder
4. Click "Commit changes"

## Deploy to Render

After uploading to GitHub:
1. Go to Render dashboard
2. Click "Manual Deploy" → "Deploy latest commit"
3. Wait 5-10 minutes
4. Your app will be live!
